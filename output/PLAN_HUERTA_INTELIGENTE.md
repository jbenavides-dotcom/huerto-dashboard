## 1. RESUMEN EJECUTIVO

### Vision del Proyecto

Transformar aproximadamente **400 m2** de la Finca La Palma y El Tucan en un **sistema agroecologico inteligente** que integre produccion de alimentos, bienestar y tecnologia de precision. El proyecto busca:

- **Autosuficiencia alimentaria** para el hotel (cocina italiana)
- **Generacion de ingresos** via Tour de Bienestar
- **Operacion optimizada** con tecnologia IoT
- **Sostenibilidad ambiental** con ciclo cerrado de nutrientes

### Numeros Clave

| Indicador | Valor |
|-----------|-------|
| Area total del proyecto | 398 m2 |
| Area de cultivo activo | 118 m2 |
| Produccion semanal estimada | 23 kg verduras + 150 huevos |
| Inversion total | USD 4,250 |
| Costo operativo mensual | COP 1,060,000 |
| Beneficio neto mensual | COP 2,390,000 |
| Periodo de recuperacion | 7 meses |
| Operario requerido | 1 persona, medio tiempo |

### Diferenciadores Unicos

1. **Huerta 100% digitalizada** con monitoreo en tiempo real
2. **Tour de Bienestar** con hierbas anticancerigenas
3. **Integracion animal** (gallinas + conejos + azolla)
4. **Ciclo cerrado** de nutrientes (cero residuos)
5. **Operacion guiada** por iPad (minima dependencia humana)

| Componente | Especificacion | Cantidad |
|------------|----------------|----------|
| Tanque principal | 1000-2000 L | 1 |
| Bomba sumergible | 1/2 HP | 1 |

| Sensor humedad suelo | Capacitivo WiFi | 3 |
| Controlador | WiFi (Tuya/Sonoff) | 1 |

### 4.3 Logica de Automatizacion

**Regla 1: Riego por humedad**
```
SI humedad_suelo < 30% durante 2 lecturas
   Y no_ha_llovido_en_24h
ENTONCES
   Generar alerta "Riego necesario"
   Crear tarea en iPad operario
   Operario activa valvula manualmente
   Sistema registra evento
```

**Regla 2: Riego invernadero (prioritario)**
```
SI humedad_invernadero < 35%
   Y temperatura > 22C
ENTONCES
   Alerta ALTA prioridad
   Riego inmediato sugerido
```

**Regla 3: Proteccion contra exceso**
```
SI humedad_suelo > 70%
ENTONCES
   Bloquear riego
   Alerta "Suelo saturado"
```

### 6.2 Proporcion de Materiales

**Compost Tradicional (C:N = 30:1)**
- 3 partes material seco (carbono)
- 1 parte material verde (nitrogeno)
- Humedad: 50-60%
- Volteo: Cada 1-2 semanas

**Lombricompost**
- Alimentacion: Cada 3-5 dias
- Espesor capa: 5-10 cm
- Humedad: 70-80% (tipo esponja)
- Cosecha humus: Cada 2-3 meses

### 6.3 Monitoreo con Sensores

| Parametro | Rango Optimo | Alerta |
|-----------|--------------|--------|
| Temperatura compost | 40-60 C | > 60 C (voltear) |
| Humedad compost | 50-60% | < 40% (humedecer) |
| Temperatura lombriz | 15-25 C | > 30 C (riesgo) |

### 6.4 Aplicacion del Compost

| Zona | Cantidad | Frecuencia | Momento |
|------|----------|------------|---------|
| Camas huerta | 2 kg/m2 | Cada ciclo cultivo | Pre-siembra |
| Invernadero | 3 kg/m2 | Trimestral | Renovacion sustrato |
| Jardin medicinal | 1 kg/m2 | Semestral | Inicio temporada |

### 7.3 Flujos de Automatizacion (IF-THEN)

**Flujo 1: Riego Huerta**
```
IF humedad_suelo < 30% (2 lecturas consecutivas)
   AND ultima_lluvia > 24h
THEN
   Alerta amarilla en iPad
   Crear tarea: "Regar Huerta - Zona A"
   Tiempo estimado: 15 min
   Operario ejecuta y marca completado
```

**Flujo 2: Estres Termico Invernadero**
```
IF temperatura_invernadero > 28C
THEN
   Alerta ROJA
   Tarea inmediata: "Ventilar invernadero"
   Mostrar camara en iPad
   Notificar propietario
```

**Flujo 3: Compost Caliente**
```
IF temperatura_compost > 60C
THEN
   Tarea: "Voltear compost"
   Advertencia: "Riesgo lombrices cercanas"
```

**Flujo 4: Agua Gallinas**
```
IF nivel_agua_gallinas < 25%
THEN
   Tarea: "Rellenar agua gallinas"
   Prioridad: Alta
```

**Flujo 5: Cosecha Programada**
```
IF dia = viernes AND hora = 6:00 AM
THEN
   Generar lista automatica:
   - Cosechar hojas (Cama 1-2)
   - Aromaticas (Cama 2 + Invernadero)
   - Recolectar huevos
   - Registrar kg en formulario
```

**Flujo 6: Auditoria Operario**
```
IF tarea_no_marcada_en_24h
THEN
   Notificar propietario
   Registrar omision
```
