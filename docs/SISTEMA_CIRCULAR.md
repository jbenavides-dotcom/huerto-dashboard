## FLUJOS DETALLADOS

### 1. HUERTA → GALLINAS
```
┌─────────────┐                      ┌─────────────┐
│   HUERTA    │ ──────────────────►  │  GALLINERO  │
│             │   Residuos:          │             │
└─────────────┘   - Hojas dañadas    └─────────────┘
                  - Restos cosecha
                  - Malezas
                  - Fruta caida
```

**Que va:**
- Hojas externas de lechuga, acelga, repollo
- Tallos de brocoli y coliflor
- Hierbas pasadas
- Cascara de verduras (del hotel)
- Malezas sin semilla

**Frecuencia:** Diaria
**Cantidad estimada:** 2-5 kg/dia

---

### 2. GALLINERO → LOMBRICOMPOST
```
┌─────────────┐                      ┌─────────────┐
│  GALLINERO  │ ──────────────────►  │ LOMBRICOMPOST│
│             │   Gallinaza:         │             │
└─────────────┘   - Estiercol        └─────────────┘
                  - Cama usada
                  - Plumas
```

**Que va:**
- Gallinaza (estiercol + orina)
- Cama usada (viruta/paja)
- Plumas
- Restos de alimento no consumido

**Frecuencia:** Semanal (limpieza)
**Cantidad estimada:** 15-20 kg/semana (25 gallinas)

**Proceso previo:**
- Dejar reposar 2-3 semanas antes de dar a lombrices
- O mezclar con material carbonado (hojas secas)

---

### 3. LOMBRICOMPOST → HUERTA
```
┌─────────────┐                      ┌─────────────┐
│ LOMBRICOMPOST│ ──────────────────►  │   HUERTA    │
│             │   Productos:         │             │
└─────────────┘   - Humus solido     └─────────────┘
                  - Lixiviado
```

**Productos:**

| Producto | Uso | Frecuencia |
|----------|-----|------------|
| Humus solido | Base de siembra, top-dressing | Cada ciclo de cultivo |
| Lixiviado (te) | Fertilizante liquido foliar | Semanal (diluido 1:10) |

**Cantidad estimada:**
- Humus: 50-100 kg/mes (2 camas activas)
- Lixiviado: 10-20 L/semana

**Aplicacion:**
- Hojas/hierbas: 1-2 kg humus/m² por ciclo
- Brasicas: 2-3 kg humus/m² por ciclo
- Tomate: 3-4 kg humus/m² por ciclo

---

### 4. AZOLLA → GALLINAS
```
┌─────────────┐                      ┌─────────────┐
│   AZOLLA    │ ──────────────────►  │  GALLINERO  │
│             │   Proteina:          │             │
└─────────────┘   - Fresca           └─────────────┘
                  - Seca
```

**Por que azolla:**
- 25-30% proteina (seca)
- Fija nitrogeno del aire
- Duplica biomasa cada 3-5 dias
- Reduce costo de concentrado

**Cantidad:**
- Fresca: 100-200 g/gallina/dia
- Meta: 25 gallinas x 150g = 3.75 kg/dia

**Manejo:**
- Cosechar 50% del tanque cada vez
- Dejar regenerar 3-5 dias
- Escurrir antes de dar a gallinas

---

### 5. COCINA HOTEL → SISTEMA
```
┌─────────────┐
│   COCINA    │ ────────┬──────────►  LOMBRICOMPOST (organicos finos)
│   HOTEL     │         │
└─────────────┘         └──────────►  GALLINAS (restos verduras)
```

**Separacion en cocina:**

| Tipo | Destino | Ejemplos |
|------|---------|----------|
| Organicos finos | Lombricompost | Cascara huevo, cafe, te, frutas blandas |
| Verduras frescas | Gallinas | Hojas, cascaras, restos ensalada |
| NO usar | Basura | Carnes, lacteos, aceites, citricos exceso |

## BALANCE DEL SISTEMA

### Entradas externas (minimizar)
- Concentrado gallinas (reducir con azolla)
- Semillas/plantulas
- Cal/minerales ocasional

### Salidas del sistema (productos)
- Verduras para hotel: ~20-25 kg/semana
- Huevos: ~150 unidades/semana
- Excedente para venta/tour

### Ciclo cerrado interno
```
SOL → HUERTA → RESIDUOS → GALLINAS → GALLINAZA → LOMBRIZ → HUMUS → HUERTA
                    ↑                                              │
                    └──────────────────────────────────────────────┘

SOL → AZOLLA → GALLINAS → GALLINAZA → LOMBRIZ → (lixiviado) → AZOLLA
         ↑                                                      │
         └──────────────────────────────────────────────────────┘
```

## CALENDARIO SEMANAL DE FLUJOS

| Dia | Actividad | Flujo |
|-----|-----------|-------|
| Lunes | Limpieza gallinero | Gallinaza → pre-compost |
| Martes | Cosecha azolla | Azolla → gallinas |
| Miercoles | Residuos huerta | Huerta → gallinas |
| Jueves | Alimentar lombrices | Organicos → lombricompost |
| Viernes | Cosecha huerta + azolla | Productos → hotel/gallinas |
| Sabado | Aplicar humus | Lombricompost → huerta |
| Domingo | Revision general | Todos los flujos |
